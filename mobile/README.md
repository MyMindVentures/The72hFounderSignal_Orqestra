# Mobile Control Center (Expo)



React Native app for monitoring autonomous build runs (Expo Router).



## Scripts



- `npm start` — Expo dev server

- `npm run web` — web target (useful with E2E against API)

- `npm test` — Jest (`jest-expo`) component tests



## API



Configure `EXPO_PUBLIC_API_BASE_URL` for your backend (see `src/lib/api.ts`). Default for Android emulator: `http://10.0.2.2:4000`.
